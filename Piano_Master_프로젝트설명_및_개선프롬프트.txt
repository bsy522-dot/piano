🎹 Piano Master — 프로젝트 설명 및 개선 요청 프롬프트
================================================================

■ 프로젝트 개요
────────────────────────────────────────
"Piano Master"는 Perfect Piano 앱을 모방한 모바일 최적화 리듬 피아노 게임입니다.
React JSX 단일 파일(piano-app.jsx)로 구현되어 있으며,
Claude Artifacts 또는 React 환경에서 바로 실행할 수 있습니다.

■ 구현된 기능 목록
────────────────────────────────────────

1. 🎹 건반 시스템
   - 5옥타브 피아노 (C2~C7, MIDI 36~96, 총 61건반)
   - 흰 건반: 통일된 흰색/연회색, 터치 시 민트색 발광
   - 검은 건반: 통일된 검정, 터치 시 보라색 발광
   - 좌우 스크롤로 원하는 옥타브 이동
   - +/- 버튼으로 줌인/줌아웃 (40%~250%)
   - 모바일 핀치 줌 (두 손가락 벌리기/오므리기)
   - 멀티터치 지원 (pointerDown/pointerUp 기반)

2. 🎵 오디오 시스템
   - Tone.js 기반 피아노 음색 합성
   - PolySynth + FM Triangle 오실레이터
   - Reverb (잔향) 효과 적용
   - 동시 16음 폴리포니 지원
   - Attack 0.005초 (즉각 반응), Decay 1.2초, Release 1.5초

3. 🎮 리듬 게임 (떨어지는 노트)
   - Waterfall 방식: 노트가 위에서 아래로 떨어짐
   - 판정 라인(초록색 가이드라인)에 노트가 닿을 때 건반을 터치
   - 판정 시스템: PERFECT (±0.12초) / GREAT (±0.22초) / GOOD (±0.38초) / MISS
   - 콤보 시스템: 연속 히트 시 콤보 증가, 10콤보마다 보너스 점수
   - 3초 리드인(준비 시간): 카운트다운 3-2-1 후 3초간 노트가 미리 내려옴
   - 노트를 놓치면 0.55초 후 자동 MISS 처리

4. 📊 채점 & 결과
   - 실시간 점수/콤보 표시
   - PERFECT/GREAT/GOOD/MISS 판정 애니메이션
   - 연주 종료 후 결과 화면: S/A/B/C/D/F 등급, 정확도(%), 최대 콤보
   - "다시 연주" / "곡 선택" 버튼

5. 🎶 수록 곡 (5곡)
   - 반짝반짝 작은 별 (쉬움)
   - 환희의 송가 - Beethoven (보통)
   - 캐논 변주곡 - Pachelbel (보통)
   - 엘리제를 위하여 - Beethoven (어려움)
   - 미뉴에트 G장조 - Bach (보통)

6. 🆓 자유 연주 모드
   - 채점 없이 자유롭게 건반 연주

7. 📱 모바일 최적화
   - 100dvh, safe-area-inset 대응 (노치, 다이나믹 아일랜드)
   - 터치 최적화 (webkit-tap-highlight 제거, overscroll 방지)
   - 스크롤바 숨김

■ 기술 스택
────────────────────────────────────────
- React (useState, useEffect, useRef, useCallback, useMemo)
- Tone.js (PolySynth, Reverb, FM Synthesis)
- CSS-in-JS (인라인 스타일)
- Pointer Events API (멀티터치)
- requestAnimationFrame (60fps 게임 루프)
- Google Fonts (Noto Sans KR, Playfair Display)

■ 노래 데이터 포맷
────────────────────────────────────────
SONGS 배열 안에 아래 형식으로 추가:
{
  id: "고유ID",
  title: "한글 제목",
  sub: "영문 부제",
  bpm: 120,
  diff: "쉬움|보통|어려움",
  dc: "#색상코드",
  notes: [
    { m: 60, t: 0, d: 0.5 },
    // m = MIDI번호 (60=C4, 62=D4, 64=E4, 65=F4, 67=G4, 69=A4, 71=B4, 72=C5)
    // t = 시작시간(초), d = 지속시간(초)
  ]
}

■ 현재 알려진 한계점 / 개선이 필요한 부분
────────────────────────────────────────
1. 피아노 소리가 합성음이라 실제 피아노와 차이가 있음 (샘플 기반이 아님)
2. 곡이 5곡밖에 없음 — 더 많은 클래식/팝 곡 추가 필요
3. MIDI 파일 임포트 기능 없음 — 수동으로 노트 데이터 입력해야 함
4. 배속 조절 기능 없음 (느리게/빠르게 연습)
5. 자동 연주(데모) 모드 없음
6. 양손(좌/우) 분리 연습 없음
7. 연주 기록 저장/재생 기능 없음
8. 떨어지는 노트와 건반 스크롤이 동기화되지만, 
   건반 바깥 노트가 렌더링되는 구조라 극단적 줌에서 간혹 이슈 가능

첨부된 piano-app.jsx 파일은 Perfect Piano 스타일의 모바일 리듬 피아노 게임입니다.
React + Tone.js로 만들어진 단일 파일 앱이며, 아래 기능이 이미 구현되어 있습니다:

- 5옥타브 건반 (좌우 스크롤, 핀치 줌, +/- 줌 버튼)
- Tone.js FM합성 피아노 소리 + 리버브
- Waterfall 떨어지는 노트 + PERFECT/GREAT/GOOD/MISS 판정
- 3초 리드인 준비시간 + 3-2-1 카운트다운
- 5곡 클래식 수록 (작은 별, 환희의 송가, 캐논, 엘리제, 미뉴에트)
- 자유 연주 모드
- 결과 화면 (등급, 점수, 콤보, 판정 통계)
- 흰건반 통일 색상 (민트), 검은건반 통일 색상 (보라)

이 앱을 더 완벽하게 개선해주세요. 아래 사항을 우선적으로 반영해주세요:

1. 곡 추가: 한국인이 좋아하는 클래식 + 2000년대 한국 가요 멜로디를 5곡 이상 추가
   (예: 학교종, 나비야, River Flows in You, 봄날은 간다 등)
2. 배속 조절: 0.5x ~ 2.0x 속도 조절 슬라이더 추가
3. 자동 연주 데모: 곡을 선택하면 "데모 보기" 버튼으로 자동 연주 시청 가능
4. 연습 모드: 틀린 노트에서 자동으로 멈추고 다시 연습할 수 있는 모드
5. 떨어지는 노트에 음이름(C, D, E...) 표시 옵션
6. 더 사실적인 피아노 소리 (Tone.js Sampler가 가능하면 적용)
7. 터치 반응 최적화: 레이턴시를 최소화하고, 누르는 순간 즉시 소리
8. 곡 진행률 프로그레스 바 추가
9. Galaxy Flip 세로 화면에서도 잘 보이도록 레이아웃 추가 최적화

기존 코드 구조와 스타일을 유지하면서 개선해주세요.
단일 JSX 파일로 유지하고, 외부 의존성은 Tone.js와 React만 사용합니다.
모바일(특히 Galaxy Flip) 최적화를 최우선으로 해주세요.
